# Question 9

## Title
Discover a Privacy Technique Used Online

## Explanation
To begin, I investigated enhanced tracking protection and private browsing as privacy techniques employed online; modern browsers can employ these techniques to prevent websites from using trackers that track user activity between different sites, helping obscure the user’s digital footprint and preventing advertisers from constructing detailed profiles of an individual’s browsing habits.
