# Question 9: Discover a Privacy Technique Used Online

## Description
I explored **Enhanced Tracking Protection** and **Private Browsing** as privacy techniques used online. Modern browsers use these methods to prevent websites from using "trackers" that follow your activity across different sites. This technique helps mask your digital footprint and prevents advertisers from building a detailed profile of your personal browsing habits.
