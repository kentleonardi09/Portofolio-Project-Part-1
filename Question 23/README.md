# Question 23

## Title
Enhance Cybersecurity at Home

## Explanation

a. Do use strong, unique passwords for each platform

Using the same password across different platforms (Wi-Fi, email, social media) is a huge security risk. It can serve as a “master key” gaining access to multiple accounts when one password has been compromised. Using varied and strong passwords on each service you access limits the potential for accounts to be compromised in the event of an external data breach.

b. Use Two-Factor Authentication (2FA)

Using two-factor authentication adds another level of security to an account so that you need more than just a password to use it. That second factor may be a mobile device, an authentication app or biometric verification. They are one of the single best ways to keep personal data secure because even if a password leaks to an outside party, 2FA makes it extremely unlikely that they would be able to access the account.

c. Keep All Devices Up to Date

Keeping every device updated with the latest software is a simple but vital cybersecurity practice. Usually updates come with security patches that help to fix identified vulnerabilities and protect them against threats of attacks. This applies to phones, laptops and even the home Wi-Fi systems that can facilitate an ongoing secure environment.
